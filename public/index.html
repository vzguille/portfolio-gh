Stacking Fault Model for Alloy Design

A cutting-edge sparsifying model is trained using high-fidelity Density Functional Theory (DFT) for Materials Design within a complex, high-dimensional compositional space on the Face-Centered Cubic (FCC) lattice structure.

This model is part of a larger project that includes more widely used Machine Learning models and features selection through Correlation and Mutual Importance. The goal of this work is to understand the complex relationships between material composition and properties, providing a more efficient method for designing new materials with desired characteristics.
Chemical Ordering in High Entropy Materials

High-Throughput DFT calculation is done using the pyiron interface in combination with the TAMU HPRC capailities. After we relax the newly generated crystallographic database we train an IAP based on the M3GNet architecture. We use this model as a calculator for thousands of new structure relaxations.

A Cluster Expansion model is fitted against this large database, then CE is used as a kernel for configuration mixing energies calculation in a supercell to study the complex alloy ordering behavior at finite temperatures using a Monte Carlo simulation.
